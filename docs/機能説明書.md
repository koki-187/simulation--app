## 1. 投資シミュレーター

### 1-1. 投資入力フォーム (/input)

投資物件の詳細情報を入力し、各種指標を自動計算するメインフォーム。

| 項目 | 範囲 | 説明 |
|------|------|------|
| **物件情報** | | |
| 物件名 | テキスト | 物件の識別名 |
| 物件価格 | ¥1M〜¥10B | 購入価格 |
| 物件種別 | 区分/戸建/一棟アパート/一棟マンション/商業 | 物件タイプ |
| **資金計画** | | |
| 諸費用 | 金額 | 購入時手数料・登記費用等 |
| 自己資金 | 金額 | 自己負担額 |
| 借入額 | 自動計算 | 物件価格 + 諸費用 − 自己資金 |
| **ローン条件** | | |
| 金利 | 0.001%〜20% | 年利率（月利率に自動変換） |
| 返済期間 | 1〜50年 | 返済年数 |
| 月返済額 | 自動計算 | 元利均等方式により計算 |
| 総利息 | 自動計算 | 返済総額 − 借入額 |
| **運営計画** | | |
| 月額家賃 | 金額 | 月間賃料収入 |
| 管理費 | 金額 | 月間管理費 |
| 修繕積立金 | 金額 | 月間修繕積立 |
| その他経費 | 金額 | 光熱費・保険等 |
| 空室率 | 0〜99% | 予想空室率 |
| 固定資産税 | 金額/年 | 年間固定資産税 |
| **減価償却** | | |
| 建物割合 | % | 物件価格のうち建物割合 |
| 構造別耐用年数 | 年数 | 鉄筋コンクリート/木造等 |
| 設備耐用年数 | 年数 | 設備の耐用年数 |
| **売却計画** | | |
| 保有年数 | 1〜50年 | 何年後に売却するか |
| 年間資産成長率 | % | 年間の物件価値上昇率 |
| **年収情報** | | |

| 申告所得ベース年収 | 金額 | 実際の申告所得 |

### 1-2. 自動計算される主要指標

#### 基本的な利回り指標

```
有効賃料 = 月額家賃 × (1 − 空室率/100) × 12
表面利回り (%) = 年間家賃 ÷ 物件価格 × 100
実質利回り (%) = (有効年収 − 年間経費) ÷ 物件価格 × 100
```

#### キャッシュフロー関連

```
年間家賃収入 = 有効賃料
年間経費 = (管理費 + 修繕積立金 + その他経費) × 12 + 固定資産税
運営CF = 年間家賃収入 − 年間経費
ローン返済額 = 月返済額 × 12
税前CF = 運営CF − ローン返済額
減価償却費 = 建物価格 ÷ 耐用年数
課税所得 = 税前CF + 給与所得 − 減価償却費
```

### 1-3. ダッシュボード（/ ）

**主要6指標の比較表示**

投資パターンごとに下記を一覧表示：
- 月返済額（万円）
- 表面利回り（%）
- 実質利回り（%）
- 1年後税引後CF（万円）
- 出口時手残り（万円）
- CAGR（%）

複数パターンを並べて比較でき、意思決定を迅速化。

### 1-4. キャッシュフロー分析（/cashflow）

30年間の年次現金フロー詳細データを表示。

| 列 | 説明 |
|----|------|
| 年 | シミュレーション年次（1〜30年） |
| 家賃収入 | 有効賃料 × 12 |
| 運営費 | 管理費・修繕積立金・その他経費の合計 |
| 固都税 | 固定資産税 |
| 運営CF | 家賃収入 − 運営費 − 固都税 |
| ローン返済 | 月返済額 × 12 |
| 税前CF | 運営CF − ローン返済 |
| 減価償却 | 建物価格 ÷ 耐用年数 |
| 課税所得 | 税前CF + 減価償却 + その他所得 − 控除 |
| 税金 | 課税所得 × 税率 |
| 税引後CF | 税前CF − 税金 |
| 累計CF | 税引後CFの累計値 |
| 残債 | ローン残高 |

**特殊機能:**
- **損益分岐点のハイライト**: 累計CF ≥ 0 となる年を自動識別・表示
- **CSV出力**: 全13列のデータをエクスポート（外部ツール連携向け）
- **PDF出力**: 横A4形式で簡潔に出力（クライアント提案資料向け）

### 1-5. 売却シミュレーション（/sale）

保有期間と価格変動シナリオを組み合わせた出口戦略分析。

| 保有期間 | 5年 | 10年 | 15年 | 20年 | 25年 | 30年 | 35年 | 40年 | 45年 | 50年 |
|---------|-----|------|------|------|------|------|------|------|------|------|

| シナリオ | 成長率 | 用途 |
|---------|--------|------|
| 悲観 | −10%/年 | リスク評価 |
| 標準 | ±0%/年 | 基本ケース |
| 楽観 | +10%/年 | ベストケース |

**表示指標:**
- 売却価格（年間成長率を反映）
- 税引前手残り（売却価格 − 残債）
- 譲渡所得税（短期/長期で区分計算）
- 税引後手残り
- 総リターン（税引後手残り + 累計CF）
- CAGR（%）
- 投資倍率（総リターン ÷ 自己資金）

**譲渡所得税の計算:**
- 取得費: 実額ルール or 5%ルール（切替可能）
- 短期譲渡（保有5年以内）: 39.63% の税率
- 長期譲渡（5年超）: 20.315% の税率

### 1-6. 融資審査指標（/ratios）

金融機関の融資判断に用いられる5つの健全性指標を表示。

| 指標 | 計算式 | 融資基準 |
|------|--------|---------|
| 年収倍率（源泉） | 借入額 ÷ 源泉年収 | 6倍以下が一般的 |
| 年収倍率（申告） | 借入額 ÷ 申告所得 | 上記に同じ |
| 返済比率（源泉） | 年間返済額 ÷ 源泉年収 × 100 | 30〜35%以下 |
| 返済比率（申告） | 年間返済額 ÷ 申告所得 × 100 | 上記に同じ |
| 表面/実質利回り | 前述の計算式 | 物件品質の判定 |

**その他の表示項目:**
- 損益分岐点賃料: 年間経費とローン返済額を賄う最低月額家賃
- DSCR（Debt Service Coverage Ratio）: 税前CF ÷ 年間返済額（1.2倍以上が理想）
- **融資健全性チェックリスト**: 5項目の合否を自動判定

### 1-7. 税金詳細計算（/tax）

不動産所得と譲渡所得の詳細税務計算を透明化。

#### 不動産所得計算
```
不動産所得 = 家賃収入 − 経費 − 減価償却費
課税対象所得 = 給与所得 + 不動産所得（−損失控除適用時）
納税額 = 課税対象所得 × 累進税率 + 住民税（10%）
```

#### 譲渡所得計算
```
譲渡所得 = 売却価格 − 取得費 − 売却費用（仲介手数料等）
納税額 = 譲渡所得 × 税率（短期39.63% / 長期20.315%）
```

**表示内容:**
- 所得税速算表（課税所得別の税率・控除額）
- 住民税（一律10%）
- 実効税率の推移（運営期間中）
- 譲渡時の税務シミュレーション

## 2. 住宅ローンシミュレーター

### 2-1. 基本フォーム（/home-sim）

住宅購入の資金計画と金利変動への対応策を検討。

| 項目 | 範囲 | 説明 |
|------|------|------|
| 物件価格 | 金額 | 購入予定価格 |
| 自己資金 | % / 金額 | 頭金 |
| 諸費用 | 金額 | 仲介手数料・登記費用等 |
| 金利タイプ | 変動 / 固定 | 変動金利 or 固定金利 |
| 金利 | % | 年利率 |
| 返済期間 | 年 | 1〜35年 |
| 年収 | 万円 | 融資審査用 |
| 税率 | % | 所得税率（金利控除の計算用） |

### 2-2. 住宅ローン控除（減税）

**制度内容:**
- 新築住宅: 13年間の控除
- 中古住宅: 10年間の控除
- 控除額: 借入残高 × 0.7% / 年（年間最大40万円）

**表示:**
- 年次別控除額テーブル（1年目〜13年目）
- 累計節税額
- 控除による実質金利の低減効果

### 2-3. 金利上昇シナリオ

変動金利選択時に、金利上昇時の返済額変化をシミュレート。

| シナリオ | 金利上昇 | 用途 |
|---------|---------|------|
| 現在 | +0% | 基本ケース |
| 軽度上昇 | +0.5% | 予備知識 |
| 中度上昇 | +1.0% | リスク評価 |
| 重度上昇 | +1.5% | 最悪シナリオ |
| 急騰 | +2.0% | エクストリームケース |

**表示:**
- 各シナリオでの月返済額（横棒グラフ）
- 返済額の増加分（視認性重視）

### 2-4. 繰上返済シミュレーション

最大3件の繰上返済イベントを設定可能。

| 項目 | 説明 |
|------|------|
| 実施年月 | 繰上返済を行う月（例: 5年目） |
| 繰上返済額 | 一度に返済する金額 |
| 返済方式 | 期間短縮型 / 返済額軽減型 |

**期間短縮型:** 返済期間が短くなり、利息が削減される。月返済額は変わらない。  
**返済額軽減型:** 月返済額が下がり、生活費に余裕が生まれる。返済期間は変わらない。

**表示:**
- 繰上返済後のローン残高推移グラフ
- 総節約利息
- 完済予定月の変化

### 2-5. 返済内訳ビジュアル

**エリアチャート:**
- 青色エリア: 元金残高の推移
- 赤色エリア: 累計支払い利息
- X軸: 返済月（1〜480ヶ月）
- 返済が進むにつれ、元金返済の比率が上昇する変化を直感的に把握

**PDF出力:** A4縦形式で、返済計画書として提案資料に使用可

## 4. 繰上返済シミュレーター

### 4-1. 基本フォーム（/prepayment）

ローンの返済加速化による効果を最大5件のイベント設定で検証。

| 項目 | 範囲 | 説明 |
|------|------|------|
| 借入額 | ¥500万〜¥2億 | 元借入額 |
| 金利 | 0.1%〜5% | 年利率 |
| 返済期間 | 5〜50年 | 返済年数 |

### 4-2. 繰上返済イベント（最大5件）

| 項目 | 説明 |
|------|------|
| 実施年 | イベント発生年（1〜返済期間） |
| 繰上返済額 | 一度に返済する金額 |
| 種別 | 期間短縮型 / 返済額軽減型 |

### 4-3. 返済方式の効果

**期間短縮型:**
- 月返済額は変わらない（負担不変）
- 返済期間が短縮される
- **利息削減効果が最大** → 総支払額が最も削減される
- キャッシュフロー重視の人向け

**返済額軽減型:**
- 月返済額が下がる（毎月の負担軽減）
- 返済期間は変わらない（完済予定は同じ）
- 生活費に余裕が出る
- 月々の現金流出を抑えたい人向け

### 4-4. 表示内容

**グラフ:**
- 折れ線グラフ: 残高推移（繰上返済後の変化を明確に）
- Y軸: ローン残高（万円）
- X軸: 経過年数

**数値指標:**
- 節約利息: 繰上返済により削減された利息総額
- 短縮期間: 期間短縮型選択時の返済期間短縮年数

## 5. 銀行・金融機関データベース

### 5-1. 住宅ローン銀行DB（/home-loan）

137商品を掲載した住宅ローン比較ツール。

**フィルター機能:**
- **エリア**: 全国 / 首都圏 / 関西圏 / 東海圏など複数選択可
- **金利タイプ**: 変動金利 / 固定金利（期限選択）
- **テキスト検索**: 銀行名・商品名で絞り込み

**表示項目:**
- 金利（色分け表示）:
  - ≤0.5%: 緑（最低金利）
  - ≤0.8%: エメラルド（低金利）
  - ≤1.2%: ネイビー（標準金利）
  - それ以上: アンバー（高めの金利）
- 月返済額試算（3,000万円借入時）
- タグ: がん保険無料 / 夫婦連帯債務対応 / ネット申込可 等

**ユースケース:** 複数銀行の金利を一覧で比較し、最適な銀行を選定

### 5-2. 投資用金融機関DB（/bank-db）

収益不動産専門の融資実績を持つ金融機関一覧。

**フィルター機能:**
- **タイプ**: 属性重視（年収・勤続年数重視） / スピード重視（短期審査） / エリア密着（地銀・信金）
- **得意構造**: 木造（W） / 鉄筋コンクリート（RC） / 重量鉄骨
- **民泊対応**: Yes / No
- **テキスト検索**: 銀行名で検索

**表示項目:**
- 金利（変動金利）
- 対応エリア（全国 / 限定エリア）
- 借入期間（構造別の最長期間）
- 最低年収要件
- 自己資金比率（20% / 30%等）
- 団信内容（一般 / 充実）
- 連絡先（電話・メール）

**ユースケース:** 投資物件の融資可否判定や金利相談先の選定

### 5-3. 提携銀行ガイド（/partner-banks）

TERASS提携・アプリ申込・書類申込の3チャネルを体系化。

### 5-4. 借入条件比較（/loan-compare）

複数の融資条件を入力し、月返済額を横並び比較。

**入力項目:**
- 借入額（万円）
- 返済期間（年）
- 返済方式（元利均等 / 元金均等）
- 年収（万円）
- 年齢（完済年齢の確認用）

**比較モード:**
- 住宅ローンモード（団信・控除を考慮）
- 収益不動産モード（税引前CFを優先）

**表示内容:**
- 月返済額の横棒グラフ（見やすく比較）
- 最大借入可能額（年収倍率・返済比率から計算）
- 完済年齢（現在年齢 + 返済期間）
- 返済比率（%） : 借入額に対する毎月返済額の割合

## 6. 資金計画書生成

### 6-1. PDF生成ツール（/funding-plan）

ワンクリックで提案用のA4縦PDF資金計画書を生成。

**入力情報:**
- 物件情報（物件名・価格・種別）
- 資金計画（自己資金・借入額）
- ローン条件（金利・返済期間）
- 運営計画（賃料・経費・空室率）
- 顧客情報（名前・住所・連絡先）

**生成内容（提案資料として利用）:**
- 投資物件概要表
- 資金計画書
- ローン返済表（初年度12ヶ月分）
- 収支シミュレーション（10年間）
- 税務計算結果

**用途:** エージェント・ファイナンシャルプランナーがクライアント提案時に即座に配付

## 7. 共通機能

### 7-1. CSV・PDF 出力

- **キャッシュフロー分析**: 30年分の詳細データをCSV形式でダウンロード
- **売却シミュレーション**: シナリオ別の出口戦略をPDF出力
- **住宅ローン計画**: 返済予定表をPDF出力
- **資金計画書**: A4縦形式の提案資料をPDF生成

### 7-2. UI/UX 設計方針

- **レスポンシブデザイン**: スマートフォン / タブレット / PC に対応
- **暗号化通信**: すべてのデータ入力は HTTPS で保護
- **データ永続化**: ローカルストレージで前回入力値を復元
- **リアルタイム計算**: フォーム入力時に即座に指標を更新
- **印刷対応**: グラフ・表がA4紙に最適化
