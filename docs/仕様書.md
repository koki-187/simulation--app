# MAS システム仕様書

## 1. システム概要

**MAS - My Agent Simulation (mas)** は、不動産投資および住宅ローンの専門的なシミュレーション機能を提供する Next.js ベースの Web アプリケーションです。2つのパターン（A/B）を同時に設定・比較でき、詳細な財務分析、税金計算、返済スケジュール、売却シミュレーション、銀行ローン比較などの高度な機能を備えています。

- **デプロイ基盤**: Railway（standalone ビルド）
- **主要な対象者**: 不動産投資家、住宅ローン検討者、FP専門家

## 4. 状態管理（Zustand Store）

### 4.1 useSimStore（投資シミュレーター用）

```typescript
type SimStore = {
  // 入力値（A パターン、B パターン）
  inputA: InputValues;
  inputB: InputValues;
  
  // 計算結果
  resultA: SimulationResult;
  resultB: SimulationResult;
  
  // アクティブパターン
  activePattern: 'A' | 'B' | 'compare';
  
  // アクション
  setInputA: (input: InputValues) => void;
  setInputB: (input: InputValues) => void;
  setResultA: (result: SimulationResult) => void;
  setResultB: (result: SimulationResult) => void;
  setActivePattern: (pattern: 'A' | 'B' | 'compare') => void;
};

interface InputValues {
  // 物件情報
  purchasePrice: number;        // 購入価格
  propertyArea: number;         // 専有面積
  buildingAge: number;          // 築年数
  
  // ローン条件
  loanAmount: number;           // 借入額
  interestRate: number;         // 金利（%）
  loanTerm: number;             // 返済期間（年）
  
  // 運営計画
  monthlyRent: number;          // 月額賃料
  operatingCost: number;        // 月額運営費
  vacancyRate: number;          // 空室率（%）
  maintenanceRate: number;      // メンテナンス率（%）
  
  // その他
  taxRate: number;              // 所得税率（%）
}

interface SimulationResult {
  monthlyNetCashflow: number;
  annualGrossIncome: number;
  annualOperatingCost: number;
  annualCashflow: number;
  totalInterestPaid: number;
  investmentReturn: number;
  // ... その他の計算結果
}
```

**永続化キー**: `mas-sim-store`  
**共有ページ**: `/`, `/input`, `/cashflow`, `/amortization`, `/compare`, `/ratios`, `/tax`, `/sale`, `/banks`, `/loan-compare`

## 6. API エンドポイント

### 6.1 金利リフレッシュエンドポイント

```
POST /api/refresh-rates
```

**説明**: Jina Reader を経由してリアルタイムに金利情報を取得し、キャッシュを更新します。

**リクエスト例**:
```json
{
  "bankName": "SBI",
  "rateType": "fixed"
}
```

**レスポンス例**:
```json
{
  "success": true,
  "data": {
    "newRate": 1.45,
    "updatedAt": "2026-05-01T15:30:00Z"
  }
}
```

**制限事項**:
- **月1回制限**: JST(日本標準時)の日付ベースで月1回のみ実行可能
- **タイムアウト**: maxDuration = 30秒
- **動的ルート**: force-dynamic
- **金利レンジバリデーション**: 0.1%～3.5% の範囲外は拒否

## 7. 計算ロジック

### 7.1 計算モジュール（src/lib/calc/）

#### refinance.ts

不動産ローンの借り換え関連計算を担当。

```typescript
/**
 * 月々の返済額を計算（元利均等）
 * @param principal 借入額
 * @param annualRate 年利（%）
 * @param months 返済期間（ヶ月）
 * @returns 月返済額
 */
export function calcMonthlyPayment(
  principal: number,
  annualRate: number,
  months: number
): number

/**
 * 借り換えの3条件をチェック
 * 1. 残期間 > 10年
 * 2. 残債 > 1000万円
 * 3. 利率差 > 0.5%
 */
export function checkThreeConditions(
  remainingMonths: number,
  balance: number,
  currentRate: number,
  newRate: number
): boolean

/**
 * 借り換えシナリオを計算（金利上昇時の試算）
 */
export function calcRefinanceScenario(
  currentRate: number,
  newRate: number,
  riseScenario: 'best' | 'moderate' | 'worst'
): ScenarioResult
```

**入力ガード**:
- principal <= 0 → 0 を返却
- months <= 0 → 0 を返却
- Infinity 上限: 999ヶ月
- breakEvenMonths は安全に並べ替え

#### simCalc.ts

投資シミュレーション全体の計算エンジン。

- 月別キャッシュフロー推移
- 年収倍率（購入価格 / 年収）
- 返済比率（月返済 / 月利息）
- 総利息、実質利回り

## 9. 主要な計算仕様

### 9.1 キャッシュフロー計算

```
月額キャッシュフロー = 
  （月額賃料 × (1 - 空室率)）
  - （月額ローン返済額）
  - （月額運営費）
  - （月額修繕費）
  + 減価償却費など
```

### 9.2 売却シミュレーション

売却年数: 5年～50年（5年刻み）

3シナリオ:
1. **強気**: 年1.5%上昇
2. **通常**: 年0.5%上昇
3. **弱気**: 年-0.5%下落

売却益 = 売却時価格 - 購入価格 - 諸費用 - 残債

### 9.3 返済スケジュール（元利均等方式）

```
残債　= 初期借入 × [ (1+月利 )^n - (1+月利)^m ] / [ (1+月利)^n - 1 ]

where:
  m = 返済済みヶ月
  n = 返済期間合計
  月利 = 年利 / 12 / 100
```
